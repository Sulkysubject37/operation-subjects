# Operation Subjects

A TUI-based bioinformatics suite that comes with all batteries included.

This application provides a user-friendly terminal interface for common bioinformatics tools. It is packaged using Conda to ensure that all required tools and dependencies are installed automatically.

## Features

- **Interactive TUI:** A clean, menu-driven interface built with `rich`.
- **Remote BLAST Search:**
    - Performs a remote BLAST search against NCBI databases (`nr`, `nt`, `swissprot`).
    - Generates a user-friendly HTML report with summary tables and plots for E-value and score distributions.
- **Phylogenetic Analysis:**
    - An automated pipeline that can start from a multi-sequence file.
    - Performs a Multiple Sequence Alignment (MSA) on the sequences using MAFFT.
    - Constructs a UPGMA phylogenetic tree.
    - Generates a final PNG image of the tree.
- **Multiple Sequence Alignment (MSA):**
    - Provides access to both `MAFFT` and `ClustalOmega`.
    - Generates a static HTML report with a conservation plot and a text view of the alignment.
- **Motif Discovery:**
    - Runs the MEME tool to discover motifs.
    - Accepts both single and multi-sequence FASTA files, and automatically applies the correct model (`-mod anr`) for single sequences.
    - Provides a link to the comprehensive HTML report generated by MEME itself.
- **Quality Control (FastQC & FastP):**
    - Runs FastQC on one or more FASTQ files to generate quality assessment reports.
    - Runs FastP to perform quality filtering, adapter trimming, and other preprocessing on both single-end and paired-end reads.
- **Genomic Annotation (Prodigal):**
    - Predicts protein-coding genes in prokaryotic genomes.
    - Outputs gene annotations (GFF), predicted protein sequences (FASTA), and predicted gene nucleotide sequences (FASTA).
    - Generates a visual HTML report summarizing gene lengths and GC content.

## Installation

1.  **Prerequisite: Install Conda or Miniconda.**
    If you don't have Conda installed, please follow the instructions on the [official Conda website](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html).

2.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd Operation_Subjects
    ```

3.  **Create the Conda Environment:**
    This command will create a new Conda environment named `operation-subjects` and install all the necessary tools and libraries.
    
    *For Apple Silicon (M1/M2/M3) users:*
    ```bash
    CONDA_SUBDIR=osx-64 conda env create -f environment.yml
    ```
    *For Intel Mac and Linux users:*
    ```bash
    conda env create -f environment.yml
    ```

4.  **Activate the Environment:**
    You must activate the environment each time you want to use the application.
    ```bash
    conda activate operation-subjects
    ```

## Usage

Once the environment is activated, you can run the application with the following command:
```bash
python main.py
```
Follow the on-screen menus to select and run the desired tool.
